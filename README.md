# Saltweb

## What Is Saltweb?

Saltweb is a weird sort of thing for hosting custom web pages. These code behind these webpages and servers are designed to be as human readable and as simplistic as possible.
Saltweb and it's features are mainly designed for, wikipedia pages, simple personal websites, portfolios, beginner web developers, or other basic web purposes.
Each page is defined as a *.sw* page, and the formatting behind them is like simpler HTML, that then gets dynamically converted to html when the server is run.
However, CSS can be used to make the sites look better, and html tags can be used in body for more advanced purposes such as buttons.

Read /docs for more information.
