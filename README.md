# SALTWEB

## What Is Saltweb?
Saltweb is a weird sort of thing for hosting custom web pages. These code behind these webpages and servers are designed to be as human readable and as simplistic as possible.\
They're mainly designed for, wikipedia pages, simple personal websites, and portfolios, or other basic web purposes.\
Each page is defined as a *.sw* page, and the formatting behind them is like simpler HTML, that then gets dynamically converted to html when the server is run.\
Read docs.md for more information.
